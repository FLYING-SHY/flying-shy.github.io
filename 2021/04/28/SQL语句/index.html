基础关键字

BETWEEN…AND 和 IN（集合）

排序查询 order by

语法：order by 子句

​ order by 排序字段1 排序方式1，排序字段2 排序方式2…

注意：
如果有多个排序条件，则当前边的条件值一样时，才会判断第二条件。

聚合函数：将一列数据作为一个整体，进行纵向的计算

分组查询

语法：group by 分组字段;

注意：分组之后查询的字段：分组字段、聚合函数

分页查询

语法：limit 开始的索引,每页查询的条数;

公式：开始的索引 = （当前的页码 - 1） * 每页显示的条数

limit 是一个MySQL”方言”
